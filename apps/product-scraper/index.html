Product Scraper

Web App · 商品スクレイパー

商品ページのURLを貼り付けると、商品名と価格を抽出して一覧表にし、CSVでダウンロードできます。複数のURLをまとめて比較表にするのに便利です。

info 取得は Cloudflare Worker（サーバー側）が行います。ページ内の構造化データ（schema.org / OGP）から抽出するため、JavaScriptで描画するサイトやbotを弾くサイトは取得できない場合があります。対象サイトの利用規約・robots.txt に配慮してご利用ください。

Result

# 商品名 価格 通貨 元

01 — URL

商品ページのURLを貼る（複数可）。

02 — Extract

Worker がページを取得し、構造化データから商品名・価格を抽出。

03 — CSV

表にまとまったら、CSVとして書き出し（Excel対応）。

Tech

Cloudflare Workers · HTMLRewriter · schema.org JSON-LD / OGP · 静的フロント (GitHub Pages)

詳細設定（Worker URL）

バックエンド Worker のURL。未設定だと動作しません。
