Hand Scroll

Chrome Extension · Live Demo

手のひらをカメラにかざして上下に動かすと、ページがスクロールします。料理中など手が離せないときの操作を想定した Chrome 拡張です。本来は拡張機能ですが、コア技術（MediaPipe による手の認識）は端末内で完結するため、この場でそのまま試せるようにしました。

lock カメラ映像は外部に送信されず、すべてあなたのブラウザ内で処理されます。

基本のパンケーキ

手が汚れていても、手をかざすだけでレシピを読み進められます。下のステップを、手のひらを上下に動かしてスクロールしてみてください。

材料（2人分）

  • 薄力粉 … 150g
  • 砂糖 … 大さじ2
  • ベーキングパウダー … 小さじ2
  • 牛乳 … 120ml
  • 卵 … 1個
  • バター … 適量

作り方

1. ボウルに薄力粉・砂糖・ベーキングパウダーを入れて混ぜる。

2. 別のボウルで卵を溶き、牛乳を加えて混ぜる。

3. 粉類に液体を加え、ダマが少し残る程度にさっくり混ぜる。

4. フライパンを弱めの中火で熱し、バターを薄くひく。

5. 生地を流し入れ、表面に気泡が出たら裏返す。

6. 両面がきつね色になったら完成。お好みでメープルシロップを。

コツ

混ぜすぎないことがふんわり仕上げる秘訣です。生地を寝かせるとさらに口当たりがよくなります。

ここまで手だけでスクロールできましたか？ これが Hand Scroll の操作感です。

— End —

01 — 起動

「カメラを起動」を押し、ブラウザのカメラ許可を承認します。初回はモデル読み込みに数秒〜（約25MB）。

02 — 手をかざす

手のひらを開いてカメラに向けるとスクロールモードに。中央が停止ゾーンです。

03 — 上下に動かす

手を上ゾーンへ動かすと上に、下ゾーンへ動かすと下にスクロール。端に近いほど速くなります。

Tech

MediaPipe Gesture Recognizer (WASM) · Chrome Manifest V3 · Side Panel API
